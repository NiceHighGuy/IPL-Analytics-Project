# IPL Analytics Project

Analyzing IPL matches and deliveries to explore:

- Top players of the match
- Team win statistics
- Score distributions

## How to Run
1. Open `ipl_analysis.ipynb` in Jupyter Notebook or JupyterLab.
2. Make sure `matches.csv` and `deliveries.csv` are in the `data/` folder.
3. Run all cells.
